遗传和变异

遗传和变异

  • 遗传 -> 亲子间的相似性
  • 变异 -> 亲子间和子代个体间的差异
    遗传和变异是普遍存在的.

性状和相对性状

  • 性状 -> 生物体所表现 形态结构/生理特征/行为方式等
  • 相对性状 -> 同种生物 同一性状 不同表现形式 不一定只有两种表现形式
  • 判定相对性状:
    1. 同种生物.
    2. 同一性状.
    3. 不同(两种及以上)表现形式.
    4. 缺一不可
