遗传和变异

遗传和变异

  • 遗传 -> 亲子间的相似性
  • 变异 -> 亲子间和子代个体间的差异
    遗传和变异是普遍存在的.

性状和相对性状

  • 性状 -> 生物体所表现 形态结构/生理特征/行为方式等
  • 相对性状 -> 同种生物 同一性状 不同表现形式 不一定只有两种表现形式
  • 判定相对性状:
    1. 同种生物.
    2. 同一性状.
    3. 不同(两种及以上)表现形式.
    4. 缺一不可

基因

  • 基因在亲子间传递的桥梁是 精子和卵细胞(生殖细胞)
  • 层次: (生殖)细胞 -> 细胞核 -> 染色体 -> DNA -> 基因 -> 性状
  • 1条染色体=1个DNA=多个基因 46条染色体=46个DNA=无数基因.
  • 性状相同,基因 不一定 相同.
  • 染色体=DNA+蛋白质.
  • 子女的每一对染色体中必有一条来自父方,另一条来自母方.
  • 生殖细胞的染色体只有体细胞中每一对染色体的一半.
  • 判断体细胞与生殖细胞:
    • 体细胞染色体成对
    • 生殖细胞染色体不成对.
  • 成对的基因位于成对的染色体上,且位置一一对应.
  • 体细胞通常具有 23对(46条) 染色体.
  • 成熟的红细胞和血小板 无细胞核和染色体.
  • 生殖细胞有 23条 染色体.

显性和隐性的判断

  • 一对相对性状杂交,后代 全 表现的性状是显性
  • 无中生有,有为隐性
  • 判断依据: 亲代是xxx,子代是xxx.

性染色体

  • 男性
    • 体细胞中性染色体:XY
    • Y染色体一定来自父方,另一条染色体来自母方.
    • 体细胞中染色体:44条(或22对)+XY
    • 精子中染色体:22条+X或22条+Y
    • 精子中性染色体:X或Y
  • 女性
    • 女性体细胞中性染色体:XX
    • X染色体可能来自父方或母方.
    • 体细胞中染色体:44条(或22对)+XX
    • 卵细胞中染色体:22条+X
    • 卵细胞中性染色体:X
  • 体细胞中与性别有关的染色体:一对
  • 生殖细胞中与性别有关的染色体:一条
  • 胎儿性别决定于:受精卵形成时
  • 生男生女取决于:哪种精子与卵细胞结合
